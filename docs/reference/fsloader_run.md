## fsloader run

Runs fsloader

```
fsloader run [flags]
```

### Options

```
  -h, --help                  help for run
      --reload-cmd string     Command to be executed when files are modified
      --watch-dir string      Dir that contains the files to be watched
      --watch-files strings   Files to be watched
```
